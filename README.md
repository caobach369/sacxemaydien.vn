# SACXEMAYDIEN Public Site

Thư mục này chỉ chứa landing page public để deploy lên GitHub Pages cho domain `sacxemaydien.vn`.

## Files

- `index.html`: landing page static
- `CNAME`: custom domain cho GitHub Pages
- `static/img/landing/hero-bg.png`: asset nền dùng trong hero section

## Deploy nhanh

1. Tạo một repository mới trên GitHub.
2. Push toàn bộ thư mục này lên branch chính của repository đó.
3. Bật GitHub Pages từ branch chính.
4. Giữ file `CNAME` để GitHub Pages nhận domain `sacxemaydien.vn`.

## Ghi chú

- Nút đăng nhập đã trỏ sẵn tới `https://app.sacxemaydien.vn`.
- Đây là bundle tách riêng khỏi code backend local.
